## ⚙️ Quick Start

### Standard Mode
```bash
# 1. Clone the repository
git clone https://github.com/<your-org>/devops-simulator.git
cd devops-simulator

# 2. Install dependencies
npm install

# 3. Configure environment variables
cp .env.example .env

# 4. Run in development mode
npm run dev
